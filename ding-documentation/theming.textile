h2. Dynamo & Subthemes

Themet er bygget op med subtheming
mothership -> _moshpit *_ -> Dynamo -> "Lokalt theme"
_ * moshpit bliver merged med dynamo senere_

*Mothership* "
pre load" theme der rytter op og fjerner / tilføjer css classer og markup til og fra drupals standard installation

*moshpit* 
et starter theme der bliver merged sammen med dynamo, når vi er kommet længere i processen ...

*Dynamo* 
bibliotekernes grund theme hvor alle elementer er defineret 

*Lokalt theme* 
Det er her de lokale biblioteker kan ændre i sites udformning


Documentation 
* subtheme documentation http://drupal.org/node/225125
* mothership: http://drupal.org/project/mothership
* moshpit: http://drupal.org/project/moshpit

h2. lokalbiblioteks version

kopier folderen "lokalbibliotek" og giv den et nyt navn - navnet skal være uden mellemrum æøå osv.
den kopierede folder plaseres i sites/all/themes/

lokalbibliotek folderen har følgende filer:
* lokalbiblotek.info 
her defineres alle de grunlæggende ting i themet, såsom regioner, theme features, CSS og Javascript filer

* style.css
basic stylesheet til overskrivning af de mest gængse elementer

* theme-settings.php
defineringer af settings til themet

* template.php
overskrivninger af elementer fra drupal.
